tmux · cheat sheet
tokyo night · leader = Ctrl+a
~/.config/tmux/tmux.conf leader = Ctrl+a
Painel = divisão dentro de uma janela. Você pode ter vários painéis abertos ao mesmo tempo numa mesma aba.
C-a+s Dividir lado a lado esquerda / direita
C-a+v Dividir empilhado cima / baixo
Alt+h Alt+l Navegar ← → entre painéis sem prefix
Alt+j Alt+k Navegar ↓ ↑ entre painéis sem prefix
C-a+H L Redimensionar ← → (5 colunas)
C-a+J K Redimensionar ↓ ↑ (5 linhas)
C-a+z Zoom no painel (fullscreen toggle)
C-a+Space Alternar layout automático
C-a+q Fechar painel sem confirmação
C-a+x Fechar painel com confirmação
Janela = aba dentro de uma sessão. Cada janela pode ter múltiplos painéis. O índice começa em 1.
C-a+c Nova janela (herda diretório atual)
C-a+, Renomear janela atual é a vírgula
C-a+& Fechar janela atual (com confirmação)
C-a+n Próxima janela
C-a+p Janela anterior
C-a+1—9 Ir diretamente para janela N
Sessão = contexto de trabalho completo (ex: "backend", "dotfiles"). Persiste mesmo depois do terminal fechar — use detach e volte depois.
dentro do tmux
C-a+S Seletor visual de sessões novo
C-a+N Nova sessão (pede nome) novo
C-a+$ Renomear sessão atual
C-a+K Matar sessão atual (com confirmação) novo
C-a+d Desanexar (detach) da sessão
C-a+( ) Sessão anterior / próxima
no terminal (fora do tmux)
tmux ls Listar sessões ativas
tmux new -s nome Nova sessão com nome
tmux attach -t nome Reanexar em sessão existente
tmux kill-session -t nome Matar sessão pelo nome
Copy mode = modo de scroll e seleção no histórico do terminal. Navegação estilo Vim. Copia para o clipboard via wl-copy (Wayland).
C-a+[ Entrar em copy mode
dentro do copy mode
hjkl Navegar (estilo Vim)
C-u C-d Meia página acima / abaixo
g G Início / fim do histórico
/ Buscar no histórico
v Iniciar seleção visual
y Copiar seleção → clipboard
Esc Cancelar / sair do copy mode
configuração
C-a+r Recarregar tmux.conf sem reiniciar
C-a+C-a Enviar Ctrl+a literal ao programa
nomenclatura
session contexto de trabalho (ex: "backend")
window aba/tab dentro da sessão
pane divisão dentro de uma janela
notas importantes
conflito C-a s sobrescreve "listar sessões" padrão. Use C-a S maiúsculo.
wayland wl-copy em copy mode. X11 → xclip, macOS → pbcopy.
plugins TPM comentado. Ative: git clone tpm && C-a I
